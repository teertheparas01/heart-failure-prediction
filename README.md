# Heart Failure Prediction using Machine Learning & Deep Learning

## 📌 Overview
This project predicts the risk of death due to heart failure using clinical data.

The model helps in early identification of high-risk patients.

---

## 📊 Dataset
- Source: Kaggle
- Records: 299 patients
- Features: 12 clinical features + 1 target variable

---

## 🧠 Models Used
- Support Vector Machine (SVM)
- Artificial Neural Network (ANN)
- Random Forest Classifier

---

## ⚙️ Workflow
1. Data Loading
2. Exploratory Data Analysis (EDA)
3. Data Preprocessing (Scaling)
4. Model Training
5. Model Evaluation
6. Model Comparison

---

## 📈 Results

| Model | Accuracy | Recall (Death) |
|---|---|---|
| SVM | 0.70 | 0.75 |
| ANN | 0.76 | 0.77 |
| Random Forest | 0.75 | ~0.75 |

---

## 🎯 Key Insight
In medical applications, **recall is more important than accuracy**, as missing a high-risk patient can be dangerous.
